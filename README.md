<h1>React Weather Search App</h1>
<p>React weather search app using API calls to get location and weather data.</p>
<h2>Technologies Used</h2>
<ul>
<li>React</li>
<li>JavaScript</li>
<li>HTML</li>
<li>SCSS</li>
</ul>

<h2>APIs Used</h2>
<ul>
<li>Visual Crossing - Weather API<br />
https://www.visualcrossing.com/weather-api</li>
<li>GeoDB - Cities API<br />
http://geodb-cities-api.wirefreethought.com/</li>
<li>ipdata - IP Geolocation API<br />
https://ipdata.co/</li>
</ul>
